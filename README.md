# Setup
Yêu cầu: Docker, Nodejs

## Setup back-end
~ cd carpark_backend

~ docker-compose up -d

~ npm install

~ npm run start
End-point: http://localhost:3333/health-check

npm run dev

## Setup front-end
~ cd carpark_frontend

~ yarn dev

#react native
npx react-native run android
npx react-native start
